# 指标说明（core/metrics.py）

本文档描述 `src/core/metrics.py` 中定义的指标，用于比较或验证稳定子表（stabilizer tableau）的质量与一致性。

## 适用对象与数据格式

- 输入：二维整数表 `tableau`，形如 `List[List[int]]` 或等价的序列结构。
- 约定：对 *n* 个量子比特，表的列数为 `2n + 1`，通常前 `n` 列为 X，后 `n` 列为 Z，最后一列为相位（模 4）。


## 指标一：exact_match

**函数**：`exact_match(a, b) -> bool`

**含义**：
- 判断两张表是否 **逐元素完全一致**。

**判定方式**：
- 将输入序列统一转为 Python 列表后，做结构与数值的全等比较。

**返回值**：
- `True`：完全一致。
- `False`：任何位置不一致，或结构不同。

**适用场景**：
- 需要严格一致性（如回归测试、精确对比）。

**注意**：
- 不做近似容差。

---

## 指标二：bit_accuracy

**函数**：`bit_accuracy(a, b) -> float`

**含义**：
- 逐元素比对两张表，计算匹配元素的比例（准确率）。

**计算公式**：

```
accuracy = matching_elements / total_elements
```

**规则**：
- 若任一表为空、或行数/列数不一致，返回 `0.0`。
- 每个元素独立比较，完全相等才计为匹配。

**返回值**：
- 匹配比例，范围 `[0.0, 1.0]`。

**适用场景**：
- 需要衡量“部分正确”时的相似度（如模型预测、近似评估）。

**示例**：
- a 与 b 共有 100 个元素，其中 90 个相等，则返回 `0.9`。

---

## 指标三：commutation_check

**函数**：`commutation_check(tableau) -> (bool, Dict[str, int])`

**含义**：
- 检查表内各行对应的 Pauli 生成子是否 **两两对易**。
- 这是稳定子表的基本一致性条件之一。

**内部计算**：
- 对任意两行 `row_i, row_j` 计算二进制辛内积：

```
sp(row_i, row_j) = Σ_k (x_i[k] * z_j[k] + z_i[k] * x_j[k]) mod 2
```

- `sp = 0` 表示对易，`sp = 1` 表示反对易。

**返回值**：
- `bool`：是否全部对易。
- `dict`：`{"violations": N}`，表示反对易对的数量。

**适用场景**：
- 验证 tableau 是否为合法稳定子生成集。

**示例**：
- 若表中有 3 对行反对易，则返回 `(False, {"violations": 3})`。

---

## 指标选择建议

- **需要严格一致性**：用 `exact_match`。
- **需要近似度量**：用 `bit_accuracy`。
- **需要物理/数学一致性检查**：用 `commutation_check`。
